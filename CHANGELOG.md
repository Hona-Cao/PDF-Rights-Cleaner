# 更新日志

## 1.1 — 2026-07-21

- 增加待处理列表和结果列表右键菜单。
- 支持两个列表的单选、多选操作。
- 支持从待处理列表拖出移除项目，不删除源文件。
- 使用 Windows Shell 原生数据对象重写结果文件拖放。
- 修复窗口自由缩放时的文字残影、色块错位和控件覆盖。
- 将界面控件统一为稳定的直角矩形风格。
- 增加原创多尺寸应用图标。
- 在“关于”中加入作者、项目主页、后续更新和 Star 提示。
- 增加 Windows 文件版本信息。

## 1.0

- 首个可用版本。
- 支持批量检测并解除无需打开密码的 PDF 所有者权限限制。
- 支持自定义缓存、批量另存和结果恢复。
