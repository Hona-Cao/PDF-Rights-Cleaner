# PDF Rights Cleaner 1.1

这是首个正式 GitHub Release。

## 主要内容

- 原生 Win32 单文件便携应用，体积小于 10 MB。
- 批量处理无需打开密码、但限制打印、复制或编辑的 PDF。
- 支持文件和文件夹拖入、列表多选、右键菜单及结果文件拖出。
- 支持可配置缓存和重启后恢复结果。
- 修复连续调整窗口大小造成的文字残影与控件错位。
- 使用 Windows Shell 原生文件对象改进跨窗口拖放。
- 加入应用图标、作者信息和 GitHub 项目入口。

## 下载与校验

下载 `PDF-Rights-Cleaner-v1.1-Windows-x64.exe` 后，可使用发布页提供的 SHA-256 校验文件验证完整性。

## 使用边界

本工具不会猜测或破解 PDF 打开密码。需要密码才能打开的文件不受支持。请只处理你拥有或获准修改的文件。
